# Estudos em Java

Repositório dedicado aos meus estudos iniciais em Java, com foco em fundamentos da linguagem e lógica de programação.

## Conteúdos praticados
- Estrutura básica de um programa Java
- Variáveis e tipos de dados
- Condicionais (if / switch)
- Laços de repetição (for / while)
- Métodos
- Introdução à orientação a objetos

## Objetivo
Consolidar a base em Java por meio de exercícios práticos e pequenos exemplos.
